# Spotify App

## Project Overview
ავაწყეთ basic დიზაინი Spotify-ს აპლიკაციისა (რეკლამების გარეშე)

## Team Contributions
| Team Member | Owned Scope |
|------------|-------------|
| გიორგი | Sidebar, Home Page |
| ანდრია | Header, Player |

## Team Decisions
### Development Approach
- პროექტი ავაწყვეთ **Desktop-First** მიდგომით, რადგან თავიდანვე გვინდოდა დავალების პირობიდან გამომდინარე პრიორიტეტი დესკტოპ ვერსია ყოფილიყო.

### Task Management
- თითოეულ წევრს ჰქონდა თავისი პასუხისმგებლობის ზონა და დავალებები წინასწარ იყო გაწერილი.

### Team Communication
- გუნდში გვქონდა **სინქრონიზაციები, მოკლე შეხვედრები, მიმოწერა**, სადაც ვათანხმებდით პროგრესს, ბლოკერებს და შემდეგ ნაბიჯებს.

### Naming Convention
- კლასების დასარქმევად გამოვიყენეთ **kebab-case, descriptive naming** მიდგომა, რათა კოდი ყოფილიყო ერთიანი და ადვილად წასაკითხი.

### Team Lead
- გუნდს ჰყავდა **Team Lead: [გიორგი]**
- მისი პასუხისმგებლობა იყო დავალებების კოორდინაცია, სტრუქტურული გადაწყვეტილებების შეთანხმება, PR-ების გადახედვა და გუნდის საერთო პროგრესის კონტროლი.

## Workflow
- ვმუშაობდით ცალკე branch-ებით.
- ცვლილებები ერთიანდებოდა pull request-ების საშუალებით.
- თითოეული წევრი პასუხისმგებელი იყო საკუთარ ნაწილზე.
